# Aplikacja Lista zadań

1. Opis

"Aplikacja Lista zadań" jest aplikacją webową umożliwiającą użytkownikowi zarządzanie zadaniami do wykonania. Użytkownik może dodawać, edytować, usuwać i oznaczać zadania jako wykonane. Aplikacja wyświetla statystyki zadań. Aplikacja działa w przeglądarce internetowej bez konieczności instalowania dodatkowego oprogramowania.

2. Wymagania funkcjonalne

- Użytkownik może dodawać nowe zadania do listy
- Użytkownik może usuwać istniejące zadania z listy
- Użytkownik może edytować treść istniejącego zadania
- Użytkownik może oznaczać zadania jako wykonane
- Użytkownik może przeglądać statystyki wykonanych zadań

3. Wymagania pozafunkcjonalne

- Aplikacja powinna zapisywać dane użytkownika lokalnie w przeglądarce
- Aplikacja powinna reagować na akcje użytkownika w czasie nie dłuższym niż 400 ms

4. Potencjalni odbiorcy systemu

- Studenci planujący naukę i zadania domowe
- Osoby pracujące, organizujące codzienne obowiązki

5. Korzyści dla użytkowników

- Ułatwienie organizacji codziennych zadań i obowiązków
- Możliwość szybkiej kontroli postępów dzięki statystykom i wizualizacji postępu

6. Stos technologiczny

HTML, CSS, JavaScript
